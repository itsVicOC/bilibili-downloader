# Changelog

## v0.2.0 - 2026-06-10

### Added

- 支持在 GUI、批量下载和 CLI 中统一解析 BV 号、AV 号、B 站完整链接与 b23.tv 短链。
- 新增 GitHub Actions 发布流水线，推送 `v*` tag 后自动构建并发布 macOS 与 Windows 压缩包。
- 新增下载可靠性测试，覆盖备用下载链接重试和断点续传行为。

### Changed

- 重设计主界面、视频信息面板、下载列表和常用对话框，提升信息密度、视觉层级和空状态体验。
- 登录、Cookie 校验、批量解析等耗时操作迁移到后台线程，减少 GUI 卡顿。
- SESSDATA 优先写入系统凭据库，凭据库不可用时回退到本地配置。
- CLI 文案更新为 `source` 输入，明确支持 BV/AV/URL/短链。

### Fixed

- 下载失败时自动尝试备用 URL，并保留可续传的临时文件。
- 取消任务时同步终止 FFmpeg 合并进程。
- 修复弹幕颜色解析和字幕序号生成问题。
