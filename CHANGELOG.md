# Changelog

本项目遵循语义化版本。日期使用 `YYYY-MM-DD` 格式。

## v0.4.1 - 2026-07-22

### Fixed

- 修复扫码登录成功后未从轮询响应读取 `SESSDATA`，导致客户端提示需手动输入 Cookie 的问题。
- 保留 SSO 重定向 Cookie 提取作为兼容回退，并更新扫码登录提示。

## v0.4.0 - 2026-07-22

### Added

- 新增多 P 单选/全选、CLI 编码与字幕语言选项，以及跨应用重启的稳定断点目录。
- 新增 macOS CI 冒烟、依赖漏洞审计、Release SHA-256 校验与 CycloneDX SBOM。

### Changed

- CLI 与 GUI 统一使用同一下载流水线，并向用户展示画质、编码回退和附属文件失败。
- 配置迁移到各平台原生目录，设置边界在创建与赋值时均强制验证。
- 主窗口支持 900×640 响应式堆叠，解析、下载与服务任务使用独立线程池。
- 重新设计滚动条、键盘焦点、复选框、设置并发步进器和登录页信息层级，并保持日间/夜间主题一致。

### Fixed

- 修正 DASH 能力位、HDR 与高级音轨识别、常见弹幕模式和多分 P 字幕查询。
- 修复登录凭据明文显示、退出登录未清理凭据、批量错误弹窗干扰和同步 FFmpeg 检查卡顿。
- 限制媒体、字幕、封面重定向目标，并限制弹幕与封面响应大小。
- 修复输出规格控件与下载按钮边框重叠、登录说明被截断及内部多余滚动条。

## v0.3.0 - 2026-07-21

### Added

- 新增 BiliFlow 二次元视觉体系：星轨控制台主界面、作品资料卡、输出规格面板、下载任务轨道及统一弹窗样式。
- 新增随操作系统实时切换的日间/夜间主题，不需要重启应用。
- 新增 Microsoft Fluent Emoji 派生应用图标，并提供 PNG、ICNS、ICO 多平台版本和可复现构建脚本。
- 新增公共素材离线打包与冻结环境资源路径解析。
- 新增登录状态后台工作线程，避免启动和凭据刷新阻塞界面。
- 新增 API、主题、配置、下载、FFmpeg、批量解析和输入校验回归测试。
- 新增用户指南、故障排查、构建发布、贡献、安全、MIT 许可证和第三方素材文档。

### Changed

- 全面重构主窗口、设置、登录、批量任务、视频信息与下载列表的交互和布局。
- 视频元数据使用最新 WBI 签名流程解析，并加强 API 返回值校验。
- 批量解析改为逐项报告结果，单项失败不再中断全部任务。
- 下载过程支持备用 CDN、Range 续传、原子目标文件处理和更准确的进度状态。
- FFmpeg 管理改进进程终止、错误输出、编码流选择与跨平台路径处理。
- 登录凭据优先写入系统凭据库，配置写入改为临时文件加原子替换。
- PyInstaller 为 macOS、Windows、Linux 分别选择 ICNS、ICO、PNG 图标，并携带全部 GUI 素材。

### Fixed

- 修复无音轨、无可用画质和不完整 API 响应可能导致的崩溃。
- 修复下载任务重试时行号变化、错误任务状态残留和取消后 FFmpeg 继续运行的问题。
- 修复临时文件在服务器不支持续传或返回异常 Content-Range 时可能损坏的问题。
- 修复多 P 文件名冲突、非法文件名字符和超长文件名处理。
- 修复 AV 号、短链、带查询参数 URL 及不支持页面类型的校验与错误提示。
- 修复设置保存失败未反馈、损坏配置未备份以及并发数越界问题。
- 修复封面加载、登录状态检查和网络操作阻塞 Qt 主线程的问题。

### Security

- SESSDATA 不再默认明文保存在配置中；系统凭据库不可用时采用受限权限文件回退。
- 配置文件默认使用当前用户读写权限，并避免非原子写入造成部分内容泄露或损坏。
- 文档明确禁止在 Issue、日志和截图中公开 Cookie 或带鉴权参数的媒体 URL。

## v0.2.0 - 2026-06-10

### Added

- 支持在 GUI、批量下载和 CLI 中统一解析 BV 号、AV 号、B 站完整链接与 b23.tv 短链。
- 新增 GitHub Actions 发布流水线，推送 `v*` tag 后自动构建并发布 macOS 与 Windows 压缩包。
- 新增下载可靠性测试，覆盖备用下载链接重试和断点续传行为。

### Changed

- 重设计主界面、视频信息面板、下载列表和常用对话框，提升信息密度、视觉层级和空状态体验。
- 登录、Cookie 校验、批量解析等耗时操作迁移到后台线程，减少 GUI 卡顿。
- SESSDATA 优先写入系统凭据库，凭据库不可用时回退到本地配置。
- CLI 文案更新为 `source` 输入，明确支持 BV/AV/URL/短链。

### Fixed

- 下载失败时自动尝试备用 URL，并保留可续传的临时文件。
- 取消任务时同步终止 FFmpeg 合并进程。
- 修复弹幕颜色解析和字幕序号生成问题。
